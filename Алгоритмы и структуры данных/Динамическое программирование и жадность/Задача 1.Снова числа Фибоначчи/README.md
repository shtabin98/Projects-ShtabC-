**Сложность по времени:**
Алгоритм выполняет один линейный проход для заполнения массива и ещё один — для вывода:
    Цикл заполнения: примерно size - 2 итераций, в каждой — константное число операций (одно сложение и запись).
    Цикл вывода: примерно size - 1 итераций, в каждой — константная операция вывода.
Итого: O(n) операций, где n=size.
**Временная сложность: O(n).**

**Сложность по памяти:**
    Используется std::vector<size_t> размера size, то есть хранится все n чисел ряда.
**Пространственная сложность: O(n).**
